Conectar con Google Sheets

Seguí estos pasos una sola vez.

1

Crear el proyecto en Google Cloud Console

  • Entrá a console.cloud.google.com
  • Habilitá Google Sheets API y Google Drive API
2

Crear credenciales OAuth

  • APIs y servicios → Credenciales → ID de cliente OAuth → Aplicación web
  • Orígenes autorizados: https://modularsoft-dbclientes.netlify.app
  • Copiá el Client ID y pegalo abajo
3

Crear la Google Sheet

  • Creá una planilla con hojas: Clientes, Facturacion, IPC, Config
  • Compartila con todos los usuarios como Editor
  • Copiá el Spreadsheet ID de la URL y pegalo abajo
4

Pantalla de consentimiento OAuth

  • APIs → Pantalla de consentimiento → Tipo Externo
  • Scopes: .../auth/spreadsheets
  • Agregá los emails del equipo como Usuarios de prueba
